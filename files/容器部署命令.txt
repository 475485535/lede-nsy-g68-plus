portainer部署

1、docker volume create portainer_data

2、docker run --restart=always -d -p 9000:9000 -v /var/run/docker.sock:/var/run/docker.sock -v portainer_data:/data -v /public:/public portainer/portainer


nginx-proxy-manager部署

1、docker volume create nginx-proxy-manager

2、docker run -d -p 8181:8181 -p 8080:8080 -p 4443:4443 --restart=always --name=nginx-proxy-manager -v nginx-proxy-manager:/config:rw  -e TZ=Asia/Shanghai   jlesage/nginx-proxy-manager
